# CareFlow Automation

## Overview
CareFlow Automation is an Excel-based automation tool designed for everyday assistance services ("Alltagshilfe").
The application simplifies administrative workflows by automatically generating invoices and service reports after entering customer data.

The main focus of the project is process automation, efficiency, and reducing repetitive manual work.

---

## Features

- Automated invoice generation
- Automatic creation of service reports ("Leistungsnachweise")
- Centralized customer data management
- Excel-based workflow automation
- User-friendly input forms
- Standardized document templates
- Reduced administrative effort through automation
- Faster processing with minimal manual interaction

---

## Technologies Used

- Microsoft Excel
- VBA (Visual Basic for Applications)
- Excel Formulas & Functions
- Automated document workflows
